Key Insights

  • Logging residues represent the largest resource category at 2.76 million dry tonnes annually, primarily concentrated in northern California counties
  • Bioenergy and electricity generation is the leading end use application, accounting for 36.6% of total potential utilization
  • Geographic concentration in Del Norte, Humboldt, and other timber-producing regions indicates infrastructure opportunities
  • Multiple market scenarios included (near-term, emerging, mature-market) provide flexibility for planning
  • High-value applications like biochemicals and materials represent emerging opportunities, especially for small-diameter trees

Data Fields

Each record in the dataset contains the following information:

  • Geographic coordinates: Latitude and longitude for spatial analysis
  • Administrative units: State, county name, and FIPS codes
  • Resource characteristics: Type, specific resource classification, and annual amount
  • Economic data: Resource price per dry tonne
  • Scenario information: Future market scenarios (near-term, emerging, mature-market)
  • Units: All resource amounts measured in dry tonnes per year
  • Metadata: Data source and URL for verification

Technical Notes

Visualization Methodology: For performance optimization, the interactive map displays the top 500 highest-concentration points for each resource category. This represents the most significant biomass sources while maintaining responsive performance.

End Use Allocation: The Sankey diagram shows potential allocation percentages based on typical biomass utilization patterns and market demands. Actual allocation would depend on infrastructure, market conditions, and policy frameworks.

Data Currency: This analysis is based on the 2023 Billion Ton Report. Users should verify current data availability and updates from the source.
